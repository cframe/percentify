# Percentify

Gem for doing percentages in ruby

2.as_percentage_of(10) = 20.0

2.percent_of(10) = 0.2

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'percentify'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install percentify
